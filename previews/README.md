# Previews — drafts work-in-progress

Carpeta para previews intermedios de blog articles. Excluida de git (gitignored).

## Por qué gitignored

- Los previews son drafts antes de migrar a `client/public/blog/`
- Una vez publicado en producción, el preview se queda aquí como referencia local
- No interesa versionar todas las iteraciones intermedias en el repo

## Cómo abrir un preview

Doble click en cualquier .html — abre en navegador. El CSS apunta a `https://caliro.dev/blog/blog-data.css` para que se vea idéntico al artículo en producción.
